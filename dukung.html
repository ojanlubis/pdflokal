Tanya Jawab

Kenapa PDFLokal gratis?

Menurut kami, tools buat urus dokumen itu harusnya gratis. Masa iya harus bayar cuma buat gabung PDF atau kompres gambar?

Kalau aku traktir kopi, fiturnya jadi berbayar nggak?

Nggak dong. Semua fitur yang ada sekarang bakal tetap gratis buat semua orang selamanya. Traktiran kopi ini buat bantu biaya domain, hosting, dan development fitur baru.

File aku aman nggak?

Aman banget. Semua proses jalan langsung di browser kamu. File nggak pernah diupload ke server manapun. Nggak percaya? Cek sendiri source code-nya ‚Äî semuanya open source.

Bakal ada fitur premium?

Nggak ada dan nggak akan pernah ada. PDFLokal ini proyek komunitas, bukan bisnis. Kalau ada fitur yang butuh server (kayak konversi Word), kita bakal cari cara biar tetap gratis.
