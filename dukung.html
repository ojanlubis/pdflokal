Pertanyaan Umum

Kenapa PDFLokal gratis?

Kami percaya tools dasar untuk mengurus dokumen seharusnya tersedia gratis untuk semua orang. Tidak ada alasan harus bayar untuk gabung PDF atau kompres gambar.

Apa donasi akan membuat fitur jadi berbayar?

Tidak. Semua fitur yang ada sekarang akan tetap gratis selamanya. Donasi membantu kami membayar domain, hosting, dan waktu development untuk fitur baru.

Bagaimana file saya aman?

Semua proses berjalan langsung di browser kamu. File tidak pernah diupload ke server manapun. Kamu bisa cek sendiri di source code - semuanya open source.

Ada fitur premium?

Tidak ada dan tidak akan pernah ada. PDFLokal adalah proyek komunitas, bukan bisnis. Kalau ada fitur yang butuh server (seperti konversi Word), kami akan cari cara agar tetap gratis.
