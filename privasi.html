Kebijakan Privasi

PDFLokal dibangun dengan satu prinsip utama: privasi kamu adalah prioritas. Semua proses edit PDF dan gambar berjalan 100% di browser kamu. Tidak ada file yang diupload ke server manapun.

Prinsip Utama Kami

  • 100% Client-Side Processing — Semua proses berjalan di browser kamu, bukan di server kami
  • Tidak Ada Upload — File PDF dan gambar kamu tidak pernah dikirim ke server manapun
  • Tidak Ada Penyimpanan — Kami tidak menyimpan file kamu di manapun, setelah tab ditutup file hilang
  • Open Source — Kode sumber terbuka, siapa saja bisa verifikasi klaim privasi kami

Cara mudah membuktikan: Coba matikan koneksi internet kamu, lalu gunakan PDFLokal. Semua fitur tetap berjalan normal karena tidak ada data yang dikirim ke server.

Data yang TIDAK Kami Kumpulkan

  • File PDF atau gambar yang kamu proses
  • Isi dokumen, teks, atau tanda tangan
  • Informasi pribadi (nama, email, dll)
  • Data lokasi atau GPS
  • Riwayat penggunaan atau aktivitas

Tentang Analytics

Kami menggunakan Vercel Web Analytics untuk memahami berapa banyak orang yang menggunakan PDFLokal. Analytics ini bersifat:

  • Anonim — Tidak ada data pribadi yang dikumpulkan
  • Agregat — Hanya statistik umum seperti jumlah kunjungan
  • Tanpa Cookie Tracking — Tidak menggunakan cookie untuk melacak kamu
  • Privacy-Focused — Tidak dijual ke pihak ketiga

Penyimpanan Lokal (Local Storage)

PDFLokal menyimpan beberapa preferensi di browser kamu menggunakan Local Storage. Data ini hanya tersimpan di perangkat kamu dan tidak pernah dikirim ke server.

Key Fungsi
pdflokal_theme Menyimpan preferensi tema (light/dark)
pdflokal_changelog_last_closed Mengingat changelog terakhir yang sudah dibaca

Keamanan

  • HTTPS — Koneksi terenkripsi untuk semua halaman
  • Security Headers — CSP, X-Frame-Options, dan perlindungan lainnya
  • No External Data Transmission — File kamu tidak pernah meninggalkan browser
  • Regular Updates — Library dan dependensi selalu diupdate

Pertanyaan?

Kalau kamu punya pertanyaan tentang privasi atau keamanan PDFLokal, silakan buka issue di GitHub repository kami.

Kamu juga bisa langsung cek source code untuk memverifikasi semua klaim privasi di atas.
